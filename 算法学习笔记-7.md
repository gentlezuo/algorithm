### 71
题目：给定一个整数 n，返回 n! 结果尾数中零的数量。  
[阶乘后的零](https://leetcode-cn.com/problems/factorial-trailing-zeroes/description/)  
分析：一个数对其进行因式分解，2的个数总是比5的个数多，10=5*2，因此字需要求出5的个数即可  
代码：
~~~
class Solution {
    public int trailingZeroes(int n) {
        int res=0;
        while(n!=0){
            n/=5;
            res+=n;
        }
        return res;
    }
}
~~~
