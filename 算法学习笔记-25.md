题目：给定一个正整数，检查他是否为交替位二进制数：换句话说，就是他的二进制数相邻的两个位数永不相等。  
[交替位二进制数](https://leetcode-cn.com/problems/binary-number-with-alternating-bits/description/)   
分析;显然位运算    
代码：
~~~
public boolean hasAlternatingBits(int n) {
        //a is    11111111..
        //a+1 is 100000000..
        int a=n^(n>>1);
        return (a&a+1)==0;
    }
